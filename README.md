# Metadata Remover iOS

Приложение для iPhone и iPad, которое помогает делиться фотографиями без лишних метаданных. Пользователь выбирает изображение из медиатеки, приложение показывает обнаруженную служебную информацию и позволяет сохранить или отправить очищенную версию.

## Возможности

- выбор фотографии через `PhotosPicker`;
- отображение ключевых метаданных (GPS-координаты, модель устройства, дата съёмки, параметры экспозиции);
- пересохранение изображения без метаданных на устройстве пользователя;
- экспорт очищенного файла в медиатеку или через системное окно "Поделиться";
- информационный экран с объяснением, почему стоит удалять метаданные.

## Стек

- SwiftUI для пользовательского интерфейса;
- PhotosUI для работы с медиатекой и выбора изображений;
- ImageIO и CoreGraphics для чтения и удаления метаданных;
- Concurrency/actors для фоновой обработки.

## Запуск

1. Откройте `MetadataRemover.xcodeproj` в Xcode 15 или новее.
2. Выберите цель **MetadataRemover** и запустите на симуляторе iOS 16+ или физическом устройстве.
3. При первом запуске предоставьте приложению доступ к Фото.

> ⚠️ Для корректной работы сохранения в Фото на симуляторе может потребоваться включить доступ к медиатеке в настройках симулятора.
